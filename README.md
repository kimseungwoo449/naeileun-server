# Naeileun
취준생들을 위한 커뮤니티 Naeileun 입니다. <br>
자신의 이력서과 자기소개서를 관리, 스터디그룹에 가입하여 다양한 정보를 공유, 채용정보를 확인, 커뮤니티 게시판을 통해 다른 사람들과 소통을 나눌 수 있는 웹 사이트 입니다. <br><br>

### 활용 기술
* Back-End
  * **JAVA**
  * **JSP**
* Front-End
  * REACT
  * CHAKRA UI
  * REACT-ROUTER
* Database
  * Amazon RDS (mySql)
